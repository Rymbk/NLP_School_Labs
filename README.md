# NLP Labs

This repository contains Natural Language Processing (NLP) labs completed as part of my academic coursework.

The labs cover key stages of an NLP pipeline, including:
- **Data preprocessing** for text datasets
- **Feature extraction and vectorization** (One-Hot, TF-IDF)
- **Word embeddings** (Word2Vec, GloVe, FastText)
- **Model training and improvement** using machine learning algorithms such as MLP
- **Transformer-based machine translation** (English–French) using the T5 architecture

The objective is to experiment with different NLP techniques and compare their performance on text classification and translation tasks.

**Technologies:** Python, Scikit-learn, Gensim, Hugging Face Transformers, Pandas, NumPy.
